> JS

- 编译
  - 优点
    - 快
  - 不足
    - 移植性不好( 不跨平台 )
- 解释
  - 优点
    - 跨平台
  - 不足
    - 稍微慢( 因为现今硬件发展较快 )

- js是单线程解释性语言( 是由于js引擎是单线程的 )
