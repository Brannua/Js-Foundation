# this 笔试题

- 函数预编译过程 this --> window

  ```
  function test() {
    console.log(this);
  }
  test();
  ```

- 全局作用域里 this --> window

- call/apply 可以改变函数运行时this指向

- obj.func(); func()里面的this指向obj

  ```
  var person = {
    name: 'xiaoming',
    sayName: function () {
      console.log(this);
      console.log(this.name);
    }
  }
  person.sayName();
  ```
